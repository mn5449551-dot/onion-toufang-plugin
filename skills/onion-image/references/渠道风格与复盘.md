## 信息流

业务实际信息流以微博粉丝通为主，复盘里多为竖版视频；图片版位规则待业务方补充。本节只作为图片 prompt 风格参考。

| 维度 | 偏好 |
|---|---|
| 节奏 | 强钩子，一眼抓注意力 |
| 色彩 | 鲜艳、高对比 |
| 元素 | 大字主标题 + 副标 + 挂件 + IP 形象 + 解题动作 |
| 文字 | 主标题字号大，占图 1/4-1/3 |
| 背景 | 暖色调、校园、卧室、书桌等熟悉场景 |
| 情绪 | 轻度卡壳→释怀，紧迫→轻松，困惑→豁然 |

典型构图：上 1/3 文字，下 2/3 IP + 题目 / 解析动态画面。

## 应用商店

| 维度 | 偏好 |
|---|---|
| 节奏 | 信息密度高，用户主动看 |
| 色彩 | 中等饱和，品牌色或暖色 |
| 元素 | 卖点 + 功能截图感 + IP + Logo + 标题 |
| 文字 | 简短直白，如“拍三遍还不对？” |
| 背景 | 干净、简洁、信任感，白底或浅渐变 |
| 情绪 | 理性、可信、解决方案感 |

典型构图：左侧文字，右侧 IP，核心功能视觉化，例如解析卡片悬浮。

## 学习机

来源：王雪纯《创意元素结构化Spec梳理 - 图文》APP 团队 / 学习机一节（飞书 wiki，2026-03-19）。

目标人群：学生群体，关注学习体验和效果。

### 品牌符号

- IP 清晰展示，突出品牌特点。
- 蓝色科技风等亮色为主色调，搭配现代感字体和界面设计。
- 强调学习机物理外观、屏幕显示效果、操作界面。

### 场景分类

| 类型 | 例 |
|---|---|
| 痛点场景 | 学生面对复杂知识点困惑、作业效率低、学习动力不足 |
| 解决场景 | 学生使用学习机专注学习、互动体验、游戏化学习过程 |
| 结果场景 | 学生成绩提升、学习自信增强、获得老师和家长认可 |

### 情绪价值

- 引发共鸣：“这道题怎么解都像在绕迷宫？”
- 激发好奇：“原来函数图像可以这么玩？”
- 建立信任：“看看和你一样的同学怎么变自信的”

### 卖点转化

| 卖点 | 可视化方向 |
|---|---|
| AI 精准学 | “专打你的知识小怪兽，不浪费一秒在已会的题上” |
| 清北名师 | “让押题大神成为你的专属教练” |
| 同步辅导 | “课前预习，课上惊艳全场” |
| 个性化学习 | “量身定制的学习剧本” |

### 人设选择

- 提分逆袭的学长/学姐：有榜样感。
- 懂你的同龄伙伴：亲切、陪伴。
- 酷而有料的老师/导师：不古板，用学生喜欢的方式讲干货。

## 4-5 月真实复盘

> 这一节来自历史渠道 / 方向表现洞察。用它提炼风格倾向，不要把任意一条旧文案当成固定模板。

| 渠道 | 跑量好的素材特点 | 下一步建议 |
|---|---|---|
| 华为 | 活泼风格突出 | 迭代开学季主题素材 |
| 小米 | 清新配色 + 简洁文案 | 测试蓝白拼图样式 |
| 荣耀 | 蓝绿紫色系稳定 | 复刻学习机活泼风格 |
| OPPO | 学习机活泼风格、蓝色双图持续跑量 | 迭代同类素材 |
| vivo | 孩子学习场景混剪 + 情绪化开头 | 测开学季主题 |
| 学习机 | NPC 老师 / 学生视角，学习机 CTR 3.96% | 扩同类 |
| 应用商店 | 对比型 / 验证性卖点，如识别准确 vs 别家 | 多做验证感表达 |
| 信息流 | 时间紧迫感，如考前 / 周末 | 强化时间节点 |

## prompt 落地

| 渠道 | prompt 关键词 |
|---|---|
| 信息流 | “强对比色彩”、“轻度卡壳情绪”、“具体场景”、“大字标题” |
