# 功能知识库：洋葱私教班

> 来源：`洋葱私教班-应用商店文案手册.md`（2026-03-06）、飞书 Wiki《洋葱私教班介绍》（最后更新 2026-04-20）和用户补充的期末备考卖点。用于 `onion-direction` / `onion-copy` / `onion-image` 理解洋葱私教班的课程事实、卖点和文案风格。不要把这里的私教班事实误写到拍题精学、同步课等其他功能里。

## 基本口径

- 产品/课程：洋葱私教班。
- 面向人群：学生。
- 投放场景：应用商店信息流广告位。
- 文案目标：让学生点进来，而不是写给家长或品牌方看的说明书。
- 核心定位：不是课程包，而是提分方案；主线是 `测评学生水平和问题 -> 定制提分方案 -> 优质学习内容、练习、答疑 -> 可视化学习结果呈现`。
- 和普通课程包的区别：普通课程包像资源库，用户自己找课；私教班像私人教练，系统帮学生排计划、盯执行、随时答疑。

## 期末备考重点

| 卖点 | 功能事实 | 适合怎么打 |
|---|---|---|
| 期末逆袭计划，限时开启 | 专属期末备考班上线，定制提分方案 + 浓缩精华课程，专为期末冲刺设计 | 临近期末，学生时间少、漏洞多、不知道从哪补；强调“不是更努力，而是更聪明地冲刺” |
| 定制备考方案，精准提分不走弯路 | 入学即做学习诊断，精准定位每科薄弱模块，再规划先补什么、后攻什么、哪些考点性价比最高 | 多科都慌、复习没方向、只会乱刷题时，强调先诊断再排路线 |
| 浓缩精华课程 | 教研团队提炼核心考点，砍掉冗余，只留当前阶段更值得学的内容 | 期末前不适合从整本书第一页慢慢熬，适合写“一节学一个重点” |
| 动态跟踪调整 | 学习过程中持续跟踪并调整路径，确保方向始终贴近当前薄弱点 | 复习计划容易跑偏、学了几天不知道有没有变好 |

## 文案结构

| 图片形式 | 字段结构 | 规则 |
|---|---|---|
| 单图 | 主标题 + 副标题 | 完整表达核心卖点，无严格字数限制 |
| 双图 | 两句排比 | 每句尽量短、顺口、有节奏 |
| 三图 | 三句排比 | 每句尽量短、顺口、有节奏 |

- 双图/三图基于单图的核心卖点方向延展，不另起炉灶。
- 每组围绕一个明确卖点方向。
- 内部生成逻辑：锁定具体场景 + 具体问题 -> 产品怎么解决 + 生动比喻 -> 生活/情绪的具体改变。

## 风格要求

要做到：

- 活泼口语化，默认学生视角，不要书面腔。
- 有冲击力，让人停下来看，不写正确但无感的废话。
- 有说服力，让学生觉得“这个对我有用”。
- 排比要节奏感强，读起来顺。
- 数字具象化优先于抽象概念，例如 40 分 / 70 分 / 90 分。
- 可以用反差对比出效果，例如“别人熬 2 小时，我学 30 分钟”。
- 问句/痛点切入优先于平铺直叙，例如“我该怎么学？”

避免：

- 太端着、太品牌向。
- 没有信息量的漂亮话。
- 双图/三图和单图方向脱节。
- 书面感重的四字成语堆砌。

## 核心卖点

| 卖点 | 功能事实 | 适合怎么打 |
|---|---|---|
| 提分闭环体系 | AI入学诊断、精准分班、每周个性化课表、学-练-测-评闭环、学情报告 | 从“不知道怎么学”转成“每天知道学哪里、怎么练” |
| 降维讲法 | 动画演示、实物实验、生活化语言、拆解难点、引导自主推导 | 把抽象知识变成能看懂的画面 |
| AI定制班 | 先测后分班，匹配最近发展区，生成专属路径 | 从“死磕不如巧学”切入，强调量身定制 |
| 30分钟沉浸课堂 | 黄金 30 分钟，课前导入、课中动画、课后练习，游戏化设计 | 对比长课压力，强调短、好懂、愿意学 |
| 全天候课后辅导 | AI快速解答 + 真人深度辅导，全时段响应，AI布置练习、自动批改、生成错题本 | 打“晚上不会谁来教”“难题不过夜” |

## 使用边界

- 私教班可以使用手册里的提分、100%紧贴校内大纲、4周特训等业务表达。
- 不把私教班的 AI分班、每周课表、AI+真人辅导写到拍题精学里。
- 不把拍题精学的“一拍识别、分步解析、继续追问”写成私教班独有能力，除非用户明确要求组合卖点。
- 用户没有指定家长视角时，默认写给学生本人。
