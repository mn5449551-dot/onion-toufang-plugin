# 文案渠道规范（信息流 / 应用商店 / 学习机）

> 出文案 skill 按当前渠道看对应章节。用户不指定人群时，**默认学生视角**；业务核心是 APP 图文，文案要让学生看见具体场景、具体问题和洋葱的产品动作。

## 全局心法

面向学生时，先做 `具体场景 + 具体情绪 + 产品动作 + 可感知变化` 的判断。学生第一视角不是强制写“我”，而是让学生看到自己的处境、语言和在意的结果。

| 目标 | 怎么写 |
|---|---|
| 锁定具体场景 + 具体问题 | 写出“晚上作业卡壳”“拍了三遍还不准”“看答案解析更懵”等能让学生觉得“是我”的瞬间 |
| 惊艳解法 | 用产品动作解释洋葱怎么解决：`洋葱一拍`、`洋葱拍题精学`、`分步解析`、`看不懂再问 AI`、`30 分钟动画课` |
| 场景奇效 | 写生活/情绪的具体变化：作业不再卡一晚、卡在哪步就问哪步、先测出薄弱点再学 |
| 产品锚定 | 每套文案至少让人知道“谁在解决、怎么解决”，避免只写“学习神器”“高效掌握” |

默认结构是：**场景问题 -> 惊艳解法 -> 场景奇效**。这是内部生成逻辑，不展示给用户。

## 信息流

> 信息流用户被动看到广告，需要强钩子打断浏览注意力，但仍要默认学生视角，不写成家长汇报或品牌宣传稿。

### 语气规则

| 特征 | 怎么写 |
|---|---|
| 钩子起手 | 主标题用疑问 / 感叹，例如“拍了改、改了拍，识别不出来？” |
| 情绪锚点 | 抓“卡壳 / 看不懂 / 紧迫 / 作业还没写完”等轻情绪 |
| 画面感 | 放进具体时刻：晚上 9 点、周末作业、考前两周、错题订正 |
| 产品动作 | 副标承接“用洋葱一拍、分步解析、看不懂再问 AI”等动作 |

### 变体节奏

3 套以上时，主标题套路要拉开：

| 套 | 主标题套路 | 例 |
|---|---|---|
| 1 | 具体卡点 | 拍了改、改了拍，识别不出来？ |
| 2 | 反问质疑 | 看完答案解析更懵了？ |
| 3 | 时间压力 | 作业还剩亿点点？ |
| 4（可选） | 旧体验 vs 新解法 | 解析又长又没重点？ |

副标题可用承接解法、加细节、结果导向三类，但必须补足产品主体和产品动作。

## 应用商店 / 学习机单图

> 应用商店和学习机的单图文案共用同一套逻辑：学生视角、口语化、有画面感、说清功能好在哪里。学习机只支持单图；应用商店保留单图、双图、三图。

### 共用单图写法

| 维度 | 规则 |
|---|---|
| 起手 | 可以直白，也可以痛点问句；不要太品牌腔 |
| 情绪 | 让学生觉得“这就是我现在遇到的问题”，不是家长视角的省心宣传 |
| 画面 | 具体到作业、错题、课堂没听懂、考前复习、晚上无人答疑 |
| 产品动作 | 优先写“洋葱一拍”“洋葱拍题精学”“先测再学”“30 分钟动画课”“AI+真人答疑”等可感知动作 |
| 结果 | 写“卡在哪步就问哪步”“薄弱点先测出来”“晚上难题不过夜”等具体变化 |

应用商店 / 学习机单图可展示为：

```markdown
**应用商店 / 学习机 · 单图**

| 编号 | 主标题 | 副标题 |
|---|---|---|
| 文案 1 | 拍三遍还不对？ | 洋葱拍题精学，一拍识别，答案解析次次对 |
```

## 应用商店双图

应用商店双图 = 两句排比或前后关系。双图必须基于同一个单图核心卖点延展，不另起炉灶。

```text
图 1：洋葱一拍，解析秒出
图 2：看不懂，再问洋葱 AI
```

自查：

- 两句要有关系：痛点 -> 解法、旧方式 -> 新方式、场景 -> 动作。
- 至少一句明确出现“洋葱”或具体产品动作，不能只写抽象结果。
- 每句要自然、顺口，不为了 8 字硬切成病句。

---

## 应用商店三图

应用商店三图 = 三句排比或递进。三句都围绕一个卖点方向，不要把三个功能硬拼在一起。

```text
图 1：题目卡壳一拍精学
图 2：归纳重点解锁大招
图 3：智能推题举一反三
```

可选关系：

- 痛点 -> 解法 -> 奇效。
- 动作 -> 过程 -> 结果。
- 功能 1 -> 功能 2 -> 总结。
- 场景 1 -> 场景 2 -> 场景 3。

至少图 2 或图 3 出现“洋葱/洋葱学园”或明确产品动作，避免用户看完记不住是谁在解决问题。

## 避免的写法

| 避免 | 为啥 |
|---|---|
| `K12 学生必备` | 太泛，没有学生画面感 |
| `高效学习神器` | 空泛，没说产品动作 |
| `妈妈省心，家长放心` | 默认学生视角下会变成家长腔 |
| `题不会做 / 拍一下 / 当场会` | 用户记不住“谁在解决”和“怎么解决” |
| 把私教班卖点写进拍题精学 | 产品事实错位 |

## 学段语气

- 小学高年级：可以更轻松、趣味。
- 初中：直接，允许一点同侪感。
- 高中：平等、干货，不哄人。
- 中职：务实，围绕职教高考、对口升学、技能证等路径。
