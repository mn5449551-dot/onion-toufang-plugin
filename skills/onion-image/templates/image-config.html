具体版位

可多选。用户只选版位，不选尺寸；卡片内展示目标尺寸、gpt 出图尺寸、压缩上限和一期支持状态。

配置

默认继承原图配置，只声明这次变化；换形式才重新确认新版位和信息分工。
角色不清时不要继续生图；回到对话确认后再保存配置。
只填大的压缩包名称来源；系统会自动生成版位文件夹和图片名。输入“方向31.zip”也会按“方向31.zip”导出，不会重复追加后缀。
每条文案在每个版位生成几套。手动输入，最多 50 套；整批最多 100 图组，超过请拆批。
选择“随机 IP”时，每个版位/每套图都由 Agent 从本地 IP 资产中重新抽取，不锁定同一个角色。
启用后生成时从字体参考库完全随机抽取，不需要人工选择具体哪张。
只有最终画面里有手机、学习机或其它电子屏幕，并且屏幕要显示可识别的洋葱 APP 功能界面时才勾选。不要在 HTML 里上传；保存后请回到 Codex 上传截图。若不展示具体 UI，可不勾选并让 Agent 弱化/模糊屏幕内容。
